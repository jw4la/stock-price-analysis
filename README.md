# Stock Price Analysis (Beginner Project)
This project downloads real stock market data using Python and performs
basic analysis including:
- Moving averages (20-day & 50-day)
- Summary statistics
- Time-series visualization

## Libraries Used
- yfinance
- pandas
- matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python stock_analysis.py

3. Enter any stock ticker (Example: AAPL, TCS.NS, INFY.NS)
